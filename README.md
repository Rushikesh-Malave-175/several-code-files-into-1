# several-code-files-into-1

Use this script to send a full project to your AI chatbot in 1 prompt.
Writes several code files into one single file.

## Structure of created file
Writes location of each file, followed by its content and followed by 3 newlines. 

### Example
```
C:\src\1.html
-content-




C:\src\2.html
-content-




C:\src\app.jsx
-content-
```
